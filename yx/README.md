# 🎵 音效合成工具

一个功能强大的 Web 版音频编辑器，支持多音轨合成、基础剪辑和音效处理。

## 功能特性

### 📁 文件管理
- 支持 WAV, MP3, OGG, FLAC 等多种音频格式
- 拖拽上传或点击选择文件
- 多音轨同时加载和管理

### ✂️ 基础剪辑
- **剪切** - 剪切选中的音频片段
- **复制** - 复制选中的音频片段
- **粘贴** - 在当前位置粘贴
- **删除** - 删除选中的音频片段
- **分割** - 在当前位置分割音频

### 🎚️ 多轨道合成
- 支持多个音轨同时播放
- 独立控制每个轨道的音量
- 静音/独奏功能
- 可视化波形显示

### 🎨 音效处理
- **音量调节** - 0% ~ 200% 增益控制
- **淡入淡出** - 0-5秒平滑过渡
- **标准化** - 自动调整音量至最佳水平
- **反转** - 反转音频播放方向
- **插入静音** - 在指定位置插入静音片段

### 💾 格式转换
- 导出为 WAV（无损）
- 导出为 MP3
- 导出为 OGG

## 使用方法

### 快速开始

1. **打开工具**
   ```bash
   # 在浏览器中打开 index.html 文件
   open index.html
   ```

   或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000

   # 使用 Node.js (需要安装 http-server)
   npx http-server
   ```

2. **导入音频**
   - 点击上传区域选择文件
   - 或直接拖拽音频文件到上传区域

3. **选择音频片段**
   - 在波形图上按住鼠标左键拖动
   - 创建选择区域进行编辑

4. **播放控制**
   - ▶ 播放
   - ⏸ 暂停
   - ⏹ 停止

### 键盘快捷键

| 按键 | 功能 |
|------|------|
| `Space` | 播放/暂停 |
| `Ctrl/Cmd + X` | 剪切选区 |
| `Ctrl/Cmd + C` | 复制选区 |
| `Ctrl/Cmd + V` | 粘贴 |
| `Delete` | 删除选区 |
| `Esc` | 清除选区 |

### 工作流程

#### 基本剪辑流程
1. 加载音频文件
2. 在波形图上选择要编辑的片段
3. 使用剪切、复制、删除等工具编辑
4. 应用音效（可选）
5. 导出最终作品

#### 多音轨合成
1. 加载多个音频文件，将自动创建多个轨道
2. 独立调整每个轨道的音量和音效
3. 使用独奏功能单独预览某个轨道
4. 导出混合后的音频

## 技术架构

### 核心技术
- **Web Audio API** - 音频处理核心
- **Canvas API** - 波形可视化
- **AudioBuffer** - 音频数据存储

## 常见问题

### Q: 为什么导出的 MP3 文件很小？
A: 当前版本主要实现 WAV 格式导出，MP3 和 OGG 格式需要额外的编码库支持。

### Q: 可以处理多长的音频？
A: 主要取决于可用内存，通常可以处理 10-30 分钟的音频。

### Q: 选择多个音频文件会怎样？
A: 每个文件会创建一个独立的轨道，可以同时播放和编辑。

### Q: 播放时有延迟？
A: Web Audio API 会有轻微延迟，这是正常现象。剪辑操作是精确到采样的。
