选择通知显示在屏幕的哪个位置

系统特性

  • 从右侧滑入的流畅动画
  • 自动堆叠，不会重叠
  • 按通知类型显示不同颜色（成功-绿色、错误-红色、信息-蓝色、警告-橙色）
  • 点击通知可手动关闭
  • 优先级系统控制显示时长
  • 鼠标悬停时暂停自动关闭
  • 进度条显示剩余时间
  • 响应式设计，支持移动端
  • 深色模式自动适配
  • 可自定义显示位置

使用示例

// 快捷方法 notify.success('操作成功！'); notify.error('发生错误！'); notify.info('这是一条信息'); notify.warning('请注意！'); notify.critical('系统崩溃！'); // 完整配置 notify.show({ type: 'success', title: '完成', message: '任务已成功完成', priority: 'high', duration: 5000 });
